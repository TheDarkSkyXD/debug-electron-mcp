# Twitch API Resources

## Official Documentation
- **[Twitch Developer Console](https://dev.twitch.tv/console)**
  - Use this to register applications and generate Client ID/Secret.
  
- **[Twitch API Reference](https://dev.twitch.tv/docs/api/reference)**
  - Comprehensive list of all available endpoints and parameters.

- **[Authentication Guide](https://dev.twitch.tv/docs/authentication)**
  - Detailed explanation of OAuth concepts, scopes, and token types.

- **[EventSub Documentation](https://dev.twitch.tv/docs/eventsub)**
  - Guide to real-time updates via Webhooks or WebSockets.

## Tools
- **[Twitch CLI](https://github.com/twitchdev/twitch-cli)**
  - Official command-line tool. Essential for testing EventSub locally and mocking API responses.
