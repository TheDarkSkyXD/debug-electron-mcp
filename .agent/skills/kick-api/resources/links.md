# Kick API Resources

## Official Documentation
- **[Kick Developer Documentation](https://docs.kick.com/)**
  - The central hub for all Kick API guides and references.

- **[KickDevDocs Repository](https://github.com/KickEngineering/KickDevDocs)**

- **[OAuth 2.1 Authentication](https://docs.kick.com/getting-started/oauth-2.1)**

## API References
- **[Channels Endpoint](https://docs.kick.com/apis/channels)**
  - Retrieve channel information and metadata.

- **[Livestreams Endpoint](https://docs.kick.com/apis/livestreams)**
  - Check stream status and get live broadcast details.

- **[Users Endpoint](https://docs.kick.com/apis/users)**
  - Fetch user profiles and identity information.

- **[Chat Endpoint](https://docs.kick.com/apis/chat)**
  - Send messages and interact with chatrooms programmatically.

- **[Webhooks & Events](https://docs.kick.com/events/introduction)**
  - Documentation on subscribing to real-time events like chat messages and stream start/end.
